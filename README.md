## What this repo contains
This repository contains a Streamlit-based discrete-event simulation for comparing SMI's OLD and NEW paint-line process structures.

## Main files
- `app.py` — current Streamlit app
- `AGENTS.md` — instructions for Codex
- `PROJECT_CONTEXT.md` — capstone and process background
- `MODEL_ASSUMPTIONS.md` — simulation logic and assumptions
- `V2_ROADMAP.md` — prioritized future improvements

## How Codex should use this repo
Any coding agent should read the project markdown files before changing the simulation.
